# Motion

Tokens de duração e curvas de easing para transições. Use via Tailwind: `duration-default ease-standard`.

## Quando usar cada duração

- **instant (0ms)** — feedback imediato (toggles internos, sem animação visível).
- **fast (100ms)** — micro-interações: hover, focus ring, ícone girando.
- **default (200ms)** — padrão para transições de cor, sombra, elevação.
- **slow (300ms)** — entradas/saídas de elementos pequenos (tooltip, popover).
- **slower (500ms)** — modais, overlays e mudanças de layout grandes.

## Curvas

- **standard** — uso geral; suaviza início e fim.
- **emphasized** — chamadas de atenção (entrada de modal).
- **decelerate** — elementos entrando em cena.
- **accelerate** — elementos saindo de cena.
- **linear** — animações contínuas (loading, progress).
