# Density

Modificadores de densidade para superfícies que precisam de versões mais compactas (tabelas, listas longas, dashboards de monitoramento).

## Três modos

- **comfortable** — padrão. Gera componentes maiores, fáceis de tocar (mobile-first).
- **cozy** — médio. Reduz altura/padding em ~10%.
- **compact** — máximo aproveitamento de espaço (~17% menor). Use em tabelas densas e ferramentas profissionais.

## Como aplicar

Densidade ainda não é controlada por um modo CSS global (como light/dark). Para superfícies densas, use tokens absolutos:

```tsx
// Tabela densa
<tr className="h-density-compact-row-min-h">…</tr>
```

> Roadmap: adicionar um data-attribute `data-density="compact"` que reescreve as variáveis de altura mínima em containers descendentes — análogo a como `data-theme="dark"` funciona hoje.
